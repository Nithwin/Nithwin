# Assets Folder

This folder contains custom graphics for the Solo Leveling themed GitHub profile.

## Files Needed

### banner.png
Create a custom banner image (1200x300px) with:
- Dark purple/blue gradient background (#1a0b2e to #16213e)
- "HUNTER NITHWIN" text in bold, stylized font
- "S-RANK DEVELOPER" badge
- Glowing particle effects

**Recommended Tools:**
- Canva (use dark gaming template)
- Photoshop/GIMP
- Figma

**Quick Option:** Use an online banner generator with Solo Leveling aesthetic

### Alternative
If you don't want to create custom graphics, the README will work perfectly with just the text-based design and existing badge images!

## Current Setup
The README is designed to work with or without custom banner images. All other visual elements use:
- Skill Icons (skillicons.dev)
- GitHub badges
- Typing SVG animation
- Profile view counter
